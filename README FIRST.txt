README file for Missile Command
-------------------------------

The following files should be bundled with this file:

BUILD.txt - instructions for building the executable.

GAME_INSTRUCTIONS.txt - instructions on how to play the game.

Java Source - the various java source and associated resources required to build and further 
    develop the game.  The java source is structured in accordance with the standard maven 2
    project archetype.

missile-command-1.0-jar-with-dependencies.jar - prebuilt binary of the game.

This game was developed on a Mac and has been tested in both Mac OS X and Windows XP.
